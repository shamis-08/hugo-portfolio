# Отчет по разработке SPA-приложения на Vue 3 с сервером на Python

## 1. Цель работы

Разработать учебное веб-приложение **Task Manager** на базе:

- **Frontend:** Vue 3 (Composition API, Vue Router)
- **Backend:** FastAPI (Python)
- **Хранение:** JSON-файл
- **Сборка:** Docker

Приложение должно реализовывать CRUD-операции над задачами, поддержку маршрутов, формы с валидацией и работу с REST API.

## 2. Постановка задачи

### Пользовательские возможности

- просмотр списка задач
- фильтрация (все / выполненные / невыполненные)
- сортировка (по дате / по названию)
- создание новой задачи
- редактирование существующей задачи
- пометка задачи как выполненной / невыполненной
- удаление задачи

### Требования к фронтенду

- Vue 3, Composition API
- компоненты: список, элемент списка, форма, хедер, футер, layout-контейнер
- маршрутизация (вложенные и именованные маршруты, 404-страница)
- использование слотов (включая scoped-слоты)
- работа с REST API
- формы с `v-model` и базовой валидацией

### Требования к бэкенду

Реализовать REST-эндпоинты:

| Метод | URL | Описание |
|------:|-----|----------|
| GET | `/api/tasks` | список задач |
| GET | `/api/tasks/{id}` | получение задачи |
| POST | `/api/tasks` | создание |
| PUT | `/api/tasks/{id}` | обновление |
| DELETE | `/api/tasks/{id}` | удаление |

Хранение — в файле `tasks.json`, перезапись при каждом изменении.
